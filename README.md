# NSCLC Clinical SAS Project

SURVIVAL ANALYSIS using NSCLC 2018 dataset (N=240)
following CDISC SDTM and ADaM standards.
Also its contains my practice code during clinical SAS class (merge, macros, sql)

---

## Dataset

- **Patients:** 240
- **Variables:** 24

## What I Did

| Phase | Work |

| SDTM | DM, DS, SC, TU, RS, FA (6 Domains) |
| ADaM | ADSL, ADTTE |
| TFL | 2 Tables + simple Listing + 1 KM Figure |
